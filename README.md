# Tennis Match Tracker PWA

Progressive Web Application для отслеживания теннисных матчей в реальном времени.

## 🎾 Функции

- ✅ Запись матчей в реальном времени
- 📊 Визуализация теннисного корта
- 👥 Управление списком игроков
- 📈 Статистика и аналитика
- 💾 Импорт/Экспорт данных
- 🎲 Генерация случайных матчей
- 📱 Работает офлайн (PWA)
- 🚀 Устанавливается на устройства

## 🛠️ Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик
- **Tailwind CSS** - стили
- **React Router** - маршрутизация
- **Vite PWA Plugin** - PWA функционал
- **LocalStorage** - хранение данных

## 📦 Установка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен сборки
npm preview
```

## 🚀 Использование

### Локальная разработка

1. Склонируйте репозиторий
2. Установите зависимости: `npm install`
3. Запустите dev сервер: `npm run dev`
4. Откройте http://localhost:3000

### Деплой

1. Соберите проект: `npm run build`
2. Папка `dist` содержит готовый статический сайт
3. Разместите на любом хостинге (Vercel, Netlify, GitHub Pages, и т.д.)

## 📱 PWA Функции

Приложение работает как полноценное PWA:

- ✅ Устанавливается на домашний экран
- ✅ Работает офлайн после первой загрузки
- ✅ Кэширует все ресурсы
- ✅ Обновляется автоматически

### Установка на устройство

**На Android:**
1. Откройте сайт в Chrome
2. Нажмите меню (три точки)
3. Выберите "Установить приложение"

**На iOS:**
1. Откройте сайт в Safari
2. Нажмите кнопку "Поделиться"
3. Выберите "На экран «Домой»"

**На Desktop:**
1. Откройте сайт в Chrome/Edge
2. Кликните на иконку установки в адресной строке
3. Или: Меню → "Установить Tennis Tracker"

## 📊 Структура данных

Все данные хранятся в LocalStorage браузера:

- `tennis-players` - список игроков
- `tennis-matches` - история матчей

### Импорт/Экспорт

Вы можете экспортировать все данные в JSON файл и импортировать их на другом устройстве.

## 🎮 Основные экраны

1. **Главный экран** - списки матчей и игроков
2. **Запись матча** - визуализация корта и запись в реальном времени
3. **Профиль игрока** - статистика и история матчей

## 🔧 Настройка

### Цвета корта

Измените цвета в `tailwind.config.js`:

```js
colors: {
  'court-green': '#2E7D32',
  'court-line': '#FFFFFF',
}
```

### PWA настройки

Измените настройки в `vite.config.ts`:

```ts
manifest: {
  name: 'Tennis Match Tracker',
  short_name: 'Tennis Tracker',
  theme_color: '#3b82f6',
  // ...
}
```
