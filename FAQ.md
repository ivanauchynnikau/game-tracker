# ❓ Часто задаваемые вопросы (FAQ)

## Установка и запуск

### Q: Как запустить проект, если npm не работает?

**A:** Есть несколько вариантов:

**Вариант 1: Docker (рекомендуется)**
```bash
docker-compose up -d
```
Откройте http://localhost:3000

**Вариант 2: Переустановка Node.js через nvm**
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install --lts
npm install
npm run dev
```

**Вариант 3: Использование готовой сборки**
Если у вас уже есть папка `dist` с собранным проектом, просто откройте `dist/index.html` в браузере или разместите на хостинге.

### Q: Минимальные системные требования?

**A:**
- Node.js 18+ (для разработки)
- Любой современный браузер для использования приложения
- 10MB свободного места
- Интернет для первой загрузки (потом работает офлайн)

## Функциональность

### Q: Где хранятся данные?

**A:** Все данные хранятся в LocalStorage вашего браузера. Это означает:
- ✅ Данные не отправляются на сервер
- ✅ Полная приватность
- ✅ Работает офлайн
- ⚠️ Данные привязаны к браузеру и домену
- ⚠️ При очистке кэша браузера данные удаляются

**Рекомендация:** Регулярно экспортируйте данные через кнопку "Экспорт всех"

### Q: Как перенести данные на другое устройство?

**A:**
1. На старом устройстве: Кнопка "Экспорт всех" → сохранить JSON файл
2. Отправить файл на новое устройство (email, облако, USB)
3. На новом устройстве: Кнопка "Импорт" → выбрать JSON файл

### Q: Можно ли использовать приложение без интернета?

**A:** Да! После первой загрузки приложение работает полностью офлайн благодаря PWA технологии.

### Q: Сколько матчей/игроков можно хранить?

**A:** LocalStorage ограничен ~5-10MB (зависит от браузера). Это примерно:
- До 1000 игроков
- До 5000 матчей
- Если достигнут лимит, экспортируйте старые данные и начните новый сезон

## PWA

### Q: Как установить приложение на телефон?

**A:**

**Android:**
1. Откройте сайт в Chrome
2. Меню (три точки) → "Установить приложение"
3. Подтвердите установку

**iOS:**
1. Откройте сайт в Safari
2. Кнопка "Поделиться" (квадрат со стрелкой)
3. "На экран «Домой»"
4. Нажмите "Добавить"

**Desktop:**
1. Chrome/Edge: иконка установки в адресной строке
2. Или: Меню → "Установить Tennis Tracker"

### Q: В чём разница между веб-версией и установленным PWA?

**A:**
- ✅ PWA открывается как нативное приложение
- ✅ Нет адресной строки браузера
- ✅ Своя иконка в списке приложений
- ✅ Быстрый запуск
- ✅ Автоматические обновления
- ✅ Работа офлайн гарантирована

## Запись матчей

### Q: Как записывать матч?

**A:**
1. Главная → "Запись нового матча"
2. Нажмите "Начать розыгрыш"
3. После розыгрыша нажмите на имя победителя
4. Повторите для следующих розыгрышей
5. "Завершить" когда матч окончен

### Q: Что такое "темп игрока"?

**A:** Среднее время розыгрышей, выигранных этим игроком (в секундах).

### Q: Что такое "темп розыгрыша"?

**A:** Среднее время всех розыгрышей в матче (в секундах).

### Q: Можно ли изменить игроков во время матча?

**A:** Да! Нажмите на иконку "Users" в правом верхнем углу экрана записи матча.

### Q: Можно ли поставить матч на паузу?

**A:** Да! Кнопка "Пауза" во время матча. Нажмите "Продолжить" для возобновления.

## Статистика

### Q: Как посмотреть статистику игрока?

**A:**
1. Главная → таб "Игроки"
2. Нажмите на имя игрока
3. Вы увидите:
   - Общее количество матчей
   - Процент побед
   - Средний темп
   - Список всех матчей

### Q: Почему статистика показывает 0?

**A:** Статистика рассчитывается только для завершённых матчей. Завершите несколько матчей, и статистика появится.

## Импорт/Экспорт

### Q: В каком формате экспортируются данные?

**A:** JSON формат. Содержит:
- Всех игроков
- Все матчи (или выбранные)
- Дату экспорта
- Версию формата

### Q: Можно ли редактировать экспортированный JSON?

**A:** Технически да, но не рекомендуется. Неправильный формат приведёт к ошибке импорта.

### Q: Что произойдёт при импорте данных?

**A:** Текущие данные будут **заменены** на импортируемые. Сделайте бэкап перед импортом!

## Проблемы и решения

### Q: Приложение не загружается

**A:**
1. Очистите кэш браузера
2. Проверьте интернет-соединение
3. Попробуйте другой браузер
4. Проверьте, поддерживает ли браузер Service Workers

### Q: Данные исчезли

**A:** Возможные причины:
- Очистка кэша браузера
- Использование режима инкогнито
- Другой браузер/профиль

**Решение:** Импортируйте последний бэкап

### Q: Приложение работает медленно

**A:**
1. Экспортируйте и удалите старые данные
2. Очистите кэш браузера
3. Проверьте, сколько данных в LocalStorage
4. Используйте современный браузер

### Q: Не могу установить PWA

**A:**
- Убедитесь, что используете HTTPS (не работает на HTTP)
- Проверьте поддержку PWA в вашем браузере
- Попробуйте обновить браузер
- Firefox не поддерживает полноценную установку PWA
