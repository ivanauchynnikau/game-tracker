# ⚡ Быстрый старт

## 🚀 Запуск за 30 секунд

### Вариант 1: Docker (проще всего)

```bash
docker-compose up -d
```

Откройте http://localhost:3000

**Готово!** ✅

---

### Вариант 2: npm (если npm работает)

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

**Готово!** ✅

---

### Вариант 3: Скрипт автозапуска

```bash
./start.sh
```

Скрипт автоматически определит лучший способ запуска.

## 📋 Что делать дальше?

### 1. Добавить первого игрока

1. Откройте вкладку **"Игроки"**
2. Нажмите **"Добавить игрока"**
3. Введите имя, например "Иван"
4. Добавьте ещё одного игрока, например "Мария"

### 2. Создать первый матч

1. Нажмите синюю кнопку **"Запись нового матча"**
2. Приложение автоматически выберет двух случайных игроков

### 3. Записать матч

1. Нажмите **"Начать розыгрыш"**
2. После розыгрыша нажмите на имя **победителя**
3. Повторяйте для каждого розыгрыша
4. Нажмите **"Завершить"** когда матч окончен

### 4. Посмотреть статистику

1. Вернитесь на главную (стрелка назад)
2. Перейдите на вкладку **"Игроки"**
3. Нажмите на **имя игрока**
4. Увидите полную статистику!

## 🎯 Основные функции

| Функция | Где найти |
|---------|----------|
| Добавить игрока | Главная → Игроки → "Добавить игрока" |

## 🔧 Проблемы?

### npm не работает

```bash
# Используйте Docker
docker-compose up -d

# Или установите nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install --lts
```

### Порт 3000 занят

**Docker:**
```bash
# Измените порт в docker-compose.yml
ports:
  - "8080:80"  # Используйте 8080 вместо 3000
```

**npm:**
```bash
# Откройте vite.config.ts
server: {
  port: 8080  # Измените на свободный порт
}
```

### Данные пропали

1. Проверьте, тот ли это браузер
2. Не использовали ли режим инкогнито
3. Восстановите из бэкапа (Импорт)
