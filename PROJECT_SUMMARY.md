# 🎾 Tennis Match Tracker - Сводка проекта

## ✅ Что было создано

Полноценное PWA приложение для отслеживания теннисных матчей на основе вашего дизайна из Figma Make.

## 💾 Хранение данных

Все данные хранятся в LocalStorage браузера:

- **tennis-players** - массив игроков
- **tennis-matches** - массив матчей с полной историей

Данные сохраняются автоматически при каждом изменении.

## 🚀 Следующие шаги

### 1. Исправление npm (если нужно)

```bash
# Установите nvm для управления версиями Node
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install --lts
nvm use --lts
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск dev сервера

```bash
npm run dev
```

Откройте http://localhost:3000

### 4. Сборка для продакшена

```bash
npm run build
```

Готовые файлы в папке `dist/`
